## ❓ About
### AJPanel is a plugin for Enigma2 based STBs (Set-Top Boxes).

<br><hr>

<!-- ============== INSTALLATION ============== -->
### 💻 Installation Script
Run this script:
```
wget https://raw.githubusercontent.com/AMAJamry/AJPanel/main/installer.sh -O - | /bin/sh
```

### ⭐ Features 

* File Manage
* IPTV
* Movies Browser
* Services/ Channels Tools
* Bouquet Editor
* PIcons Manager
* EPG Tools
* Terminal Emulator
* SoftCam Tools
* Plugins
* Backup/Restore
* Date/Time
* Network
* Context-Menu -> Channels Browser
* Context-Menu -> Find
* Signal Monitor
* Custom Menu
